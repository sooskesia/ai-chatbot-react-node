# AI Chatbot (React + Node.js)

A simple AI-powered chatbot application with a React frontend and a Node.js backend.  
The backend integrates with OpenAI's Chat Completion API to generate conversational responses.

---

## Features

- User-friendly chat interface built with React
- Sends user messages to backend API
- Backend calls OpenAI API to get AI-generated replies
- Displays bot responses in the chat window
- Simple, minimal, and extendable architecture

---
